# AI Log Analyzer Tool v1

A Python-based AI Log Analyzer to detect suspicious patterns and summarize system logs.  
Author: LordAbdulla

---

## Features

- Load and analyze log files  
- Summarize logs: INFO, WARN, ERROR counts  
- Detect repeated/suspicious patterns  
- Keyword-based log filtering  
- Alert when threshold exceeded
